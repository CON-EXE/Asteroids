# Asteroids

## Context

This started as a guided project done as a part of Boot.devs Backend Developer Path. 

At the end of the guided part it contained a simplified yet functional game loop with basic visuals, the game didn't reset on player death and the program just closed, and there was no scoring system.

Basically it had the bare minimum needed for something to be called Asteroids.

Everything added after the addition of the README was done by myself and will be listed below.
